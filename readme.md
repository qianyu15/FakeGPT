# FakeChatGPT Pro

Offline rule-based ChatGPT-like interface built with pure HTML, CSS, and JavaScript.

No API. No network. No machine learning.
Just deterministic logic pretending to understand you.

---

## 🧠 Overview

FakeChatGPT Pro is a lightweight browser-based chatbot UI that mimics a modern AI assistant interface.

Instead of using real AI models, it relies on simple rule-based responses and keyword matching.

It exists somewhere between:
- UI experiment
- educational demo
- digital illusion of intelligence

---

## ✨ Features

- 💬 ChatGPT-like chat interface
- ⚡ Instant offline responses (no server required)
- 🧠 Simple memory system (session-based)
- ⌨️ Typing animation for AI responses
- 🎭 "Thinking..." indicator for simulated reasoning
- 🔤 Basic keyword-based response engine

---

## 📁 Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript (no frameworks, no dependencies)

---

## 🚀 How to Use

1. Download or clone the repository
2. Open `index.html` in your browser
3. Start chatting

No build step. No installation. No patience required.

---

## 🧩 How It Works

The system uses a simple rule-based function:

- Input text is matched against keywords
- Predefined responses are returned
- If no match is found, a random fallback message is shown

There is no natural language understanding.
Only pattern matching and controlled randomness.

---

## 🧠 Example Behaviors

| Input | Output |
|------|--------|
| こんにちは | Greeting response |
| 名前 | Self-identification |
| 覚えて | Stores message in memory |
| 思い出して | Shows recent messages |
| anything else | Random fallback response |
