How it works

The project is intentionally simple. It stores session messages in memory, checks user input for keywords, and returns a matching response or a fallback message.

1
Input arrives The user types a message into the chat box. A deeply advanced human ritual.
2
Rules are checked The script searches for words like “こんにちは”, “名前”, “覚えて”, or “思い出して”.
3
Response appears A predefined answer is displayed with a typing effect so it feels less like a spreadsheet.
